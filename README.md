# Presentation 
API Blog est une API complète de blog. Mon objectif était de creer un API en apprennant de nouveaux concepts tel que : le test, rateLimit, logs et les appliquer à ce projet

# Fonctionnalités
- Authentification (Inscription et connexion)
- Gestion de catégorie
- Gestion d'article

# Techologies utilisées
- Nodejs (Express)
- Prisma ORM
- POSTGRESQL
- API REST

# Installation

## Clone le projet
`https://github.com/TraoreAllassane10/API-BLOG.git `

## Installation des dépendances
`npm install `

## Démarrage du projet
`npm run dev`

## Lancement des tests
`npm test`
